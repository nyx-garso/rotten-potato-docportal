## Functional Description
The **Sign Up** module allows new users to create an account securely.  
Key features include:
- Input fields for email, username, and password.
- Validation for required fields and password strength.
- Email verification to confirm account ownership.
- Secure storage of credentials using encryption.

---

## Use Case Scenario

| Actor        | Action                                                                 | System Response                                      |
|--------------|------------------------------------------------------------------------|------------------------------------------------------|
| User         | Navigates to the Sign Up page                                          | System displays registration form                    |
| User         | Enters email, username, and password                                   | System validates input fields                        |
| User         | Clicks "Sign Up"                                                       | System creates account and sends verification email  |
| System       | Sends confirmation link to user’s email                                | User receives email and clicks verification link     |
| User         | Confirms email                                                         | System activates account and redirects to homepage   |
